### 2.3.2. User Task Matrix

**Segmento #1: Solicitante de Servicios**

| Tarea                         | Frecuencia    | Importancia      |
|-------------------------------|----------------|----------------|
| Buscar profesionales | Alta   | Alta   |

| Realizar pagos por el servicio | Alta    | ALta   |

| Coordinar fechas o entregas | Media  | Media  |
